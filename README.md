Installation
------------

```bash
npm install --save refill-next-handler
```

API
---

### Constructor

```JavaScript
var RefillNextHandler = require('refill-next-handler');

var refillNextHandler = new RefillNextHandler(options)
```

#### options

configuration object

##### Default values

```JavaScript
{
  handleSuccess: true,
  ignoreFailures: false,
  quickFinish: false,
  watch: false
}
```
